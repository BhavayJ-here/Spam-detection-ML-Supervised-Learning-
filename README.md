# Spam-detection-ML-Supervised-Learning->

A spam detection system that explores confidence, uncertainty, and model failures instead of only focusing on accuracy.

Confidence Aware Spam Detection

A machine learning spam classifier built to understand not only what the model predicts, but also how confident it is and where it fails.

Features

• Spam classification using TF IDF features

• Confidence based routing

• Uncertainty detection for borderline messages

• False positive and false negative analysis

• Model performance evaluation

Results

Accuracy: 99%

Spam Precision: 100%

Spam Recall: 89%

Spam F1 Score: 94%

False Positives: 0

False Negatives: 16

Why I Built It

Most spam projects stop at accuracy. I wanted to explore model confidence, uncertainty, and failure analysis to better understand when a prediction should be trusted.
